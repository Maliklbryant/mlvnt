# MLVNT Assets

## malik.jpg — REQUIRED

Place your portrait photo here:
  src/assets/malik.jpg

The "Meet Malik" avatar imports this file directly:
  import malikImg from "./assets/malik.jpg";

Requirements:
- Filename: malik.jpg  (exact — case sensitive)
- Format: JPEG
- Minimum size: 400×400px recommended
- Face should be roughly centered in the frame

Vite will bundle and fingerprint this file at build time.
The hashed filename (e.g. malik.a1b2c3d4.jpg) will be
placed in dist/assets/ automatically. No /public needed.
